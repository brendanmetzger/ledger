Make sure you understand the following principles:

When do you need an event listener?
Simply put, you need an event listener when you want to plan around something happening, but you don’t know when, where when falls anywhere from any moment to never.
What’s up with the word callback?
First, it is a regular function.
A regular function becomes a callback when it is passed to another function as a parameter.
The function where it has been sent to becomes responsible for executing it—that is important—you do not use the () operator after the function name.
