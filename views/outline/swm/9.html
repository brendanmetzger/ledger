Listening

The most fundamental concern of an interactive designer is the ability to react to situations. Sometimes it is an existential situation like a device resolution. More often a developer chooses to create situations to interact with, and this is done through an understanding of DOM events.

Review

Last week we covered some important CSS properties fundamental to the concerns of an overlay. We wrote the entirety of a cosine function using nothing but functions and simple mathematics that we all have possessed (perhaps forgotten) since Junior High.

Getting ready

Open this week’s practice html and css files, as well as the global JavaScript file. In the stylesheet, give the body 20% padding on the left. Create a fixed nav element with a width of 20% and a height of 100vh. Give it a background-color. In the nav put a ul, and four li elements inside. In each li put a a with the text one, two, three and four. The href attribute should be directed to #one, #two, #three and #four respectively. Below the nav place four div elements each with an id of one, two, three and four respectively. Put some text in each element as well. In your stylesheet, set the div elements to have a height of 100vh and give each div a different background color.
