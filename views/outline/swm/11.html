Case Study: Lightbox and JSON

Plainly spoken from json.org, JavaScript Object Notation) is a lightweight data-interchange format. It is easy for humans to read and write. It is easy for machines to parse and generate. It is based on a subset of the JavaScript Programming Language.

This is a great thing to get a taste of; imagine you can create and add data to a project without having to author the data, or write the HTML over and over. Many feeds and services all over the internet can be gathered and applied in this manner. Here I have used a concept called JSONP to gather images from some subreddits and display them for a lightbox.
