Review

Last week we learned about scope and made a mouse trail. Here are some thinking activities to get warmed up.

Consider the function appendChild
What object does it belong to? What does it want as a parameter? What does it return?
Consider the function createElement
What is it a method of? What parameter do you supply? What does it return?
Consider the function addEventListener
What is it a method of? What parameters do you supply? Tell me extra about the second parameter.
Considering return statements..
Obviously you should know where and how to look return values in documentation. What is another way to find out what a function returns?
