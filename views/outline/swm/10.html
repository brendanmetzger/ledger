Review

We have been discussing functions with more confidence every week. We know about declaring them, applying them, passing them undeclared to other functions as callbacks. You should be thinking about functions as nothing more than another type of variable; let’s revisit what variables are and what values they hold.

Lest we forget, there are a handful of types that can represent a value in JavaScript. First are primitive types, such as Number, String, Boolean, Null and Undefined. Then, there is one other non-primitive type called Object. Objects are combinations of primitive types as well as other objects, and we can make an object literal in a fashion such as var obj = {};. The Array is a type of Object as is the Function. Yes, that's right, in JavaScript we call a function a callable object.
