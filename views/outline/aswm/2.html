Methods

Functions are largely about encapsulating process. Sometimes that means breaking down a much bigger routine into smaller, more manageable chunks. Other times it simply defines a set of instructions that must be repeated at often or at unknown times.

Warming up

We understand types and flow of control with loops and conditions. Moving forward I've devised a few examples to showcase some areas for you to focus on in the near future.

Part One

  1. Define a containing Element
  2. Place 5-10 Elements in the containing Element
  3. Each of the 5-10 elements should be the full height of the viewport
  4. Add one more additional Element after the containing Element—as a sibling
  5. Give that last element a class attribute of hidden, and hide the element

Part Two

  1. Give each of the 5-10 elements a random background color from an array of background colors. There should be the same number of background colors in the array as there are Elements in your container.
  2. research the properties scrollTop, scrollY, scrollHeight and innerHeight—note which or what sort of object those properties belong to.
  3. Add an event listener to monitor page scroll and observe that behavior
  4. Calculate the percentage you have scrolled
  5. if the user has scrolled a reasonable percentage, remove the hidden class attribute of the element that has that attribute.
