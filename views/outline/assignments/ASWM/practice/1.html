For Next Week

Moving Forward...

Now that your course site has (hopefully) been set up, you can manipulate the HTML and CSS as you like. Before you do, there are some things I want you to be hyper-critical of:

  • Do not delete any files, the basic course structure (html, css, js) must remain intact so I can find your assignments without you having to go out of your way to submit anything.
  • If you add files, make that clear through links on your index page and additions to sitemap.xml.
  • Keep your HTML, JS, and CSS as organized as possible. Run your html through a validator if you think it might be broken. Always close your empty tags (img, link, br, etc.), and make sure assets always have the correct path.
  • Maintain a global stylesheet and global css file that you use in every project (it should already be linked). The goal is to work on programs incrementally, so hopefully extra effort in some weeks will continue to benefit you in the long term.
