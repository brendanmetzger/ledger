On the JavaScript side

Conversely, you will use JavaScript to describe the exact same dataset. The learning goal is to see the juxtaposition of describing information in two different formats, their strengths and weaknesses.

The object you chose to outline in html can be replicated, outlined and declared as a JavaScript object. The notation is fairly flexible but you may have to go about building your object in a trial and error fashion. To accomplish it that way you should have your browser console open, and you should work iteratively in practice-4.js.
