Practice Assignment

This week the goal is to use articles, sections, etc. to create the outline of a document. I gleaned some stuff from wikipedia and put together a plain text outline that you can use as content for this exercise. Note: when gathering that content, you will need to view source to do the cutting and pasting.

Your Task

  • Once you have the content, paste it into the html > body > main Element of your SWM/practice/1/index.html file
  • Referencing the Outlining an HTML document above, start organizing the HTML
  • Once marked up, apply text styles and color to give your markup some aesthetic properties
  • Upload your work to the IAM server so I can see it.

Moving Forward...

Now that your course site has (hopefully) been set up, you can manipulate the HTML and CSS as you like. Before you do, there are some things I want you to be hyper-critical of:

  • Do not delete any practice/project files; the basic template structure (html, css, js) must remain intact so I can find your assignments without you having to go out of your way to submit anything.
  • If you add files, add links and additions to sitemap.xml so I can find them.
  • Keep your HTML, JS, and CSS as organized as possible. Run your html through a validator if you think it might be broken. Always close your empty tags (img, link, br, etc.), and make sure assets always have the correct path.
  • Maintain a global stylesheet and global css file that you use in every project (it should already be linked). The goal is to work on programs incrementally, so hopefully extra effort in some weeks will continue to benefit you in the long term.
