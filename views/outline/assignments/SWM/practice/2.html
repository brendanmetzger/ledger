Introduction to Programming

Lynda.com has a great introduction to programming (general programming), but coincidentally beneficial to us, it uses JavaScript as the example language to illustrate the process and requirements. This points to the power of the language as both a useful learning tool, and as a great language to illustrate the important parts and the caveats of programming languages that some higher-level languages even avoid. it’s highly relevant, and pretty friendly for beginners.

Starting here, I want you to carefully watch chapters 1–4 this week. It is about an hour and a half, but it is slow and careful and will make the rest of the semester worthwhile. (We will be going up to chapter 11). In the videos that follow, you will see the speaker use some files, namely a container.html file and a script.js and he uses default text editor. You will be using your practice/2/index.html, practice/2/scripts/practice-2.js and your own text editor. You've already corrected all of your file paths, so you can begin working with your JavaScript file right away.

Checklist:

  • In practice/2/scripts/practice-2.js you should be following along with the videos.
  • Use comments to keep notes within your code and separate chapters and exercises.
  • The author makes extensive use of the alert function. I highly suggest you swap out console.log and get familiar with what it is doing (plus, all those dialogs are annoying and don’t keep a visual record of output).
  • If you have questions, add them to README.txt. If you have notes you want to keep track of throughout the semester, add them to the README.
  • If you have nothing to write in your readme, delete the contents so I know you had nothing to say.
  • Every little bit of code should be organized, my nitpicking will get stricter throughout the semester.
