The Web Development Workflow

Today's lecture is titled mis en place, a French culinary phrase which means everything in its place, because it is the one thing every student can easily achieve that will guarantee a positive impact in any profession involving precision and craft. Having the proper resources and tools (and learning their correct usage) will both reduce the amount of mistakes and increase the amount of joy you feel with your efforts. I sincerely promise, learning a few tools efficiently will change the trajectory of your time in front of a computer forever.

The Web Designer/Developers Fundamental* Toolkit

* Mind, I say fundamental for a reason—there are many more tools you are likely become familiar with, ranging from media and graphics editors, to online tools, to reference materials and tutorial sites.

  • A text editor, or more accurately, a Source Code Editor
  • A modern web browser (I suggest Google Chrome as it will be used in class and to evaluate all coursework)
  • An FTP Client
  • The Web Development console; built into modern browsers and we will use the hell out of it.
  • The W3C Validators (Markup and Cascading Style Sheets)
  • Awareness of documentation and resources (Such as The Mozilla Developer network [preferred] and W3Schools)
  • Typing profiency (or practice if needed), and the learning of keyboard shortcuts.
  • Understanding Software Documentation Documentation is very much a tool, and reading and writing it are skills to acquire.
  • As far as this class is concerned, ake sure you poke around Lynda.com—I'll reference it frequently. I watch it myself quite a bit and you should too!
