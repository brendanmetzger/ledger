Layout Modes

Layout modes are the algorithms that determine the position, size, and interaction of boxes especially in regard to their siblings and ancestor boxes.

The Block Layout
Strategies for designing the layout capable of columns in terms of containers and their relationships. This technique is done primarily through the float property.
Strategies and skills necessary to achieve block layouts are fundamental
The Inline Layout
Designed for layout out text—think about standard typographic things, line-height, letter-spacing, font-size etc.
Inline layouts are fundamental
The Table Layout
If you are doing anything that would store well in a spreadsheet, this is your layout tool of choice.
The Positioned Layout
This layout mode is for positioning elements primarily through concepts of a coordinate system.
Used to make relationships between individual elements less meaningful—for instance, if you wanted to overlap items in layers you would adjust their position property.
The Flexible Box Layout
Flexible layouts allow the designer to take their mind off of the details and let the platform or device determine layout sizes and positions.
Good for responsive layouts that fit traditional patterns, such as a grid of images where the specific sizes are not a concern.
Has an inverted learning curve: easier to implement basics, intermediate to complicated task require much more knowledge.
The Grid Layout
Designed for laying out elements relative to a fixed grid.
Very new.
