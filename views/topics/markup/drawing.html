Drawing

There are two fundamental ways to draw using web technologies: using the canvas element and with Scalable Vector Graphics referred to as SVG. To draw with the canvas element, you start with an html element called canvas and from there, you must use a JavaScript API to make shapes, set fill colors, etc. SVG on the other hand, is an XML file, and these files can be constructed by vector drawing programs like Adobe Illustrator, embedded as an image like any other format (<img src="file.svg"/>) or embedded directly into an HTML page as an <svg> element
