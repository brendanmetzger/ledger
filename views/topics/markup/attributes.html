Attributes

So far our HTML has been pretty basic so far in that we have not investigated attributes unless they were mandatory (like the href in anchors. An attribute is always written on the opening tag of an Element and is defined by a keyword followed by an = sign and a quoted string of text. W use them to start describing unique characteristics that might apply to a specific node and features it has. For instance, in the case of an <a> Element, the href attribute (known as the hypertext reference) determines where the anchor should take the user when they engage it.

Attributes are now an essential part of our vocabulary as we will be learning and using new ones throughout the semester. Some attributes apply to only specific Elements, while others are universal, in that we can apply them to any element.
