Smooth Scrolling

I've written a smooth-scrolling snippet that you can use if you like. This bit of code, like the way of JavaScript-based animation in general, will go the way of the dinosaur soon. Like transitions and keyframe animation, there is already a CSS spec on the books to facilitate a transitioned scroll… the countdown begins. >For data on support for the css-scroll-behavior feature across the major browsers you can check caniuse.com

While investigating anchors, you should find some interest in the :target pseudo-class. Note above, text of the <h2> will be red if the document has #smooth-scrolling at the end of the url—very handy.
