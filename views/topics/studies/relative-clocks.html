A utility Function

Often functions really just need to convert some data type into another, in this case it turns a string into a number.

The variable table serves as dictionary object that stores values against a string key. Here, we take advantage of that and map human times like day or hour into their numerical equivalent.

The Regular Expression

Here is a Regular Expression (RegEx) to convert the string to a more consistent array. Reading from the left, the expression indicates: "match 1 or more digits, maybe a period and 0 or more digits followed by spaces and one of the following words: day, hour, minute or second" Mars' string of "24 hours 39 minutes and 35 seconds" converts to the Array ['24 hour', '39 minute', '35 second']

A doozy of a function

This is a function that takes a function as a callback parameter and returns a function. You may never quite wrap your head around this, but if you do, you will experience about 5 days of euphoria. Give or take.

Here is what it does: give it an integer for a start position, the total hours, and something to do—a callback—on each cycle. Hypothetically, say we had an alarm clock that needed to go off at 8am, and the cycle—or day—is reset every 24 hours. That code would be clock(8, 24, (buzzz));

The last piece is the clock itself—a simple SVG element I made. If svg is unfamiliar, spend some time researching it—it’s very handy, and can be used as a vector image, as data and you can interact with it with both JS and CSS.
