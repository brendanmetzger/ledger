First-Class Functions
Either accept a function as an argument or return an function as a value. This concept is extremely useful for code abstraction. JavaScript employs first class functions regularly.
Pure Functions
A function that does not mutate values or situations outside of the function itself. A higher level concept important to functional programming
Recursion
Algorithms that operate in an iterable fashion by evaluating only the input and whether or not it should execute itself again with the modified value of the input.
