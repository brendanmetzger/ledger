Prototypes

JavaScript is a prototypal language, which is also known as class-less or instance-based programming. Unlike class-based languages that define relationships in a familial way (inheritance, parents, traits), prototypal languages share their genes by cloning prototypes.

It is a rather peculiar concept, but objects have a property called the prototype and objects can copy each others prototypes.
