Selectors: Type and Descendent

The most basic form of selectors are called type selectors where you use the name of the specific element to target it. The snippit p { properties }, would select all <p> elements; the snippit body { properties } would select all <body> tags (but remember, there is only one per document).

Type selectors are typically very broad, but luckily we will learn to become infinitely precise using stylesheet syntax. The first way we'll add precision and specificity is through descendant selection. We are already understanding the role of an element as an object, and now we will enforce the relationship an element object has with other element objects.
