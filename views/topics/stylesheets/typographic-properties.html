Introductory Properties: Typography

It has been written that Web Design is 95% typography and giving some weight to that claim is the amount of leverage you have for controlling the typographic experience.

Though we will do a more thorough investigation later, it would be a good idea to familiarize yourself with CSS units—you really only need to get comfortable with a handful of them and understanding how they work..

font-size property
Font-size sets the size of the text. Other properties can be based off of this size; when you declare a font-size for a particular element, that size will govern how all other properties are set when using a unit such as em
font-family property
Changes the typeface of an element.
You can embed fonts, or use system fonts. If using system fonts, separate the fonts with a comma to have defaults in place if the user does not have the specified font.
When specifying a font with multiple words, make sure to surround the name in double or single quotes.
font-weight property
Controls weight thin/normal/bold, or in 3 digit numbers like 200, 400, 800 (preferred, especially for full typefaces with more than three weights)
Mind that if you are bolding text, it is advisable to use the <strong> or <b> tags to lend semantic meaning
font-style property
Can make italicized and oblique text (if your typeface has such nuances).
Mind that if you are italicizing text, it is advisable to use the <em> or <i> tags to lend semantic meaning.
line-height property (leading)
Usage is line-height: (num)unit;
Unlike print typography dominated by baselines, web typography is dominated by this unit. Mostly, you will use to control leading, or the space between lines.
line-height need not be a fixed number, it can also be a unit-less ratio such as 1.5 which would represent 1.5 times the value of whatever the font-size is—handy!
letter-spacing property (tracking)
Adjusts the space between letters, can be positive or negative—mind, this is a different concept from kerning.
text-align property
Used to align left, center, right align, as well as justify text. You can only align text in block-level elements.
text-transform property
Offers transform to upper/lowercase, can capitalize a sentence.
text-decoration property
Controls underline, line-through, overline, etc.
text-indent property
set the indentation using a CSS unit.
You can set a negative number for a hanging indent.
Dregs
word-spacing like letter spacing, but for words.
font-variant controls advanced typographic features such as small-caps and ligatures (when available)
font-kerning controls the usage of kerning information (when available)
text-shadow can create a drop shadow for text, or an embossed effect that was very popular in 2009
Many more features, such as columns, how to deal with white space, word breaks, etc.
