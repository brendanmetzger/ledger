CSS Units: length and size

The most notable CSS units are involved in determining length. Consider the sample boxes below each have their width property set to the same number with a different unit. Make sure to resize your window to understand how some units are affected by the size of your browser (vw/vh) or the size of their containing element (%) or the size of the font (em). Pixels are static and not affected proportionally, but be aware that modern screens have different pixel sizes, so scaling and approximation will occur and designing with pixels is becoming unfavorable especially where mobile and responsive design are concerned.

Also, spend some time looking at margins and padding. Picture a bunch of cardboard boxes on the floor; the is the space between the boxes is the margin, and within each box the space afforded the contents is called the padding.
